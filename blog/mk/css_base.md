## CSSを実装するには

CSSは言語なので当然記述する必要がありますね。  
**style.css**というファイルを準備します。  
それを各HTMLファイルで以下のコードで読み込みます。  
```
<link rel="stylesheet" href="/css/style.css">
```

CSSファイルは今後も用途ごとに増えたりするでしょうし、専用フォルダがあるほうが整理しやすいでしょう。  

## CSSの記述

さて、では次に作成したstyle.cssに中身を記載していきます。  
今回やることは以下の通りです。  
- サイト全体フォントの設定
- 横幅の制限と余白の設定
- 見出し、段落の体裁の調整

デザインまでは踏み込まず、読みやすさの向上を目指す形ですね。  

### サイト全体フォントの設定

body要素を指定することでサイト全体に設定を反映させることができます。  
今回は最低限ということでこの2つだけにしておきます。  
- font-family：フォント
- line-height：行間

### 横幅の制限と余白の設定

article要素は以下の内容を指定しました。  
- max-width：最大の横幅
- margin：余白

これらを指定することで記事が真ん中に配置されます。  

### 見出し、段落の体裁の調整

article要素のh2、h3、pの前に少し余白を追加します。
- margin-top
- margin
