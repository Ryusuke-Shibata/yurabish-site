## CSSを読み込む

CSSを使うには、まずスタイルを記述するファイルを用意します。  
今回は**style.css**というファイルを準備します。  
そして各HTMLファイルで以下のように読み込みます。  
```
<link rel="stylesheet" href="/css/style.css">
```

今後CSSファイルが増えることも考えて、今回はcssフォルダを作ってそこに配置しています。  
用途ごとに整理しやすくなるのでおすすめです。  

## CSSの記述

次に作成したstyle.cssに中身を記載していきます。  
今回やることは以下の通りです。  
- サイト全体フォント設定
- 横幅の制限と余白の設定
- 見出し・段落の余白調整

デザインまでは踏み込まず、まずは読みやすさの向上を目指す形ですね。  

### サイト全体フォントの設定

body要素を指定すると、ページ全体に設定を反映させることができます。  
今回は最低限としてこの2つだけにしておきます。  
- font-family：フォントの種類
- line-height：行間

### 横幅の制限と余白の設定

記事本文にはarticle要素を使っているので、以下の内容を指定しました。  
- max-width：記事の最大横幅
- margin：余白

max-widthで横に広がりすぎないように、  
margin: autoで記事をページ中央に配置します。  

### 見出し、段落の余白調整

最後に、記事内の以下3つの要素の余白を調整します。  
- h2
- h3
- p

それぞれにmarginやmargin-topを設定し、見出し・段落の前後に適度な余白を作っています。
