## 作業

### 画像の置き場所の作成

まず記事の中に表示をする画像の置き場所を作成します。  
imagesフォルダを作ってその中に記事ごとにフォルダを作成するのがいいのかな、と思っています。  
記事が増えていくと管理が大変になるような気もしますが、  
普段は便利なツールを使って画像投稿をしているので、本来はこういうことをしないといけないんですね。  
投稿を続けていって、改善できそうなところは改善していこうと思います。

### 画像の参照

置き場を準備した後はそれを参照する必要があります。  
まずHTMLで記載するとこんな感じです。  

【HTML】
```
<img src="画像のパス"
     alt="画像の説明文">
```

そして、これをMarkdownで書くとこんな感じです。  

【Markdown】
```
![画像の説明文](画像のパス)
```

……、ということで。  
特に画像の方を準備していなかったので取り急ぎ準備しました。  
マウスでSairyHubのロゴを書いてみました。  
![SairyHubロゴ 案](/img/image_01/SairyHub_logo.png)  

高校生くらいの時はこういうカリグラフィー的な文字を書くのが好きでした。  
そのうちちゃんとしたロゴにしたいですね、とりあえず原案ということで。
