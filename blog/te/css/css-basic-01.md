## CSSの読み込み

CSSを読み込む方法は3つあります。

---

### 方法①：外部CSS
HTMLファイルの<head>タグ内でCSSファイルを読み込む方法です。    
```html
<link rel="stylesheet" href="style.cssのパス">
```
link要素の**href属性**はいわゆるハイパーリンク、外部のリンク先を指定します。  
そして、**rel属性**はそのリンクとの関係性を指定します。  
つまり上記の場合は、**「style.cssはstylesheetである」**ということを記載しています。  
基本的に何か理由がないのであればこの方法を使うのがベストです。  

---

### 方法②：内部CSS
HTMLファイル内に<style>タグを直接記載する方法です。  
```html
<style>
  body {
    color: red
  }
</style>
```
この方法の場合、HTMLファイルごとにCSSが動作するので検証用途がメインとなる。  

---

### 方法③：インラインCSS
タグなどに直接記載する方法です。  
```html
<p style="color: red;">
  テキスト
</p>
```
要素ごとにCSSが動作するので非推奨。

## 基本設定の実装（フォント・横幅・余白など）

まず、細かい話の前に基本的基本文法を押さえておきましょう。  
```css
body{
  color: red
}
```
bodyを**セレクタ**、colorを**プロパティ**、redを**値**と呼びます。  
それぞれ、以下のような役割があります。  
- セレクタ：適用先
- プロパティ：変更するもの
- 値：変更内容

1つのセレクタに対し、複数のプロパティを設定することもありますが、その時はセミコロンでつなげましょう。  

さて、それではブログ作成に必要になりそうな基本設定の実装方法を確認しておきましょう。  
### ①　フォント設定
```css
body {
  font-family: Arial, sans-serif;
}
```
複数指定することでフォントの表示に不具合が生じた場合に備えておく。  

### ②　横幅の制御
```css
.container {
  max-width: 1200px;
  margin: 0 auto;
}
```
margin: 0 auto; は中央寄せ。

### ③　余白の基本
```css
body {
  margin: 0;
  padding: 0;
}

.section {
  padding: 40px 20px;
}
```
paddingは内側、marginは外側の余白を指定する。

### ④　幅と高さの計算方法
```css
* {
  box-sizing: border-box;
}
```
幅（width）にborder、paddingを含める設定で、%指定でレイアウトするなら必須。  
初期値はcontent-boxで、こっちは含めない。

そのほかにもありますが、網羅して覚えるようなものでもないので割愛。  

---

## 重要ポイントまとめ
- 外部CSSで実装するのがベター
- CSSは土台から指定していく（body、containerなど）
- 「box-sizing: border-box;」の指定は必須
