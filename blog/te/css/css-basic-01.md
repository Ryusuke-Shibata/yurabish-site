### CSSの読み込み

CSSを使うには、まずそのスタイルを記述するファイルを用意する必要があります。  
名前は何でもいいんですが、style.cssという名前を使っているケースが多いですかね。  
そして、それを各HTMLファイルで以下のようにして読み込みます。  
```html
<link rel="stylesheet" href="style.cssのパス">
```
link要素の**href属性**はいわゆるハイパーリンク、外部のリンク先を指定します。  
そして、**rel属性**はそのリンクとの関係性を指定します。  
つまり今回の場合は、**「style.cssはstylesheetである」**ということを記載しています。

### 基本設定の実装（フォント・横幅・余白など）

まず、細かい話の前に基本的基本文法を押さえておきましょう。  
```css
body{
  color: red
}
```
bodyを**セレクタ**、colorを**プロパティ**、redを**値**と呼びます。  
それぞれ、以下のような役割があります。  
- セレクタ：適用先
- プロパティ：変更するもの
- 値：変更内容

1つのセレクタに対し、複数のプロパティを設定することもありますが、その時はセミコロンでつなげましょう。  

さて、それではブログ作成に必要になりそうなフォント・横幅・余白あたりのプロパティを確認しておきましょう。  
まず、フォントは**font-family**で指定します。  
フォントを指定するだけなら**font**でいいのではないか、と思いますが、そこにfamilyがついているのには理由があります。  
このプロパティは**利用するフォント名をカンマで区切って記述する**ものとなっています。  
書いたものは左から順にフォントを探し、見つかったものから使っていくという動きをします。  
閲覧する環境によっては指定したフォントが見つけられない・利用できない可能性もあるので、  
このようにいくつか指定するのがいいとされています。  
ちなみにフォント名に空白を含むものもありますが、その時は引用符で囲むようにしましょう。
