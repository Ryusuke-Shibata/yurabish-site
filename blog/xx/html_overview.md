## 前提知識など

HTMLがタグによって記述する言語であることはもともと知っていました。  
また、Webサイトがディレクトリ構造を持っていてそれがURLとなる、といったような  
かなりふわっとした理解は大学の講義で触れた記憶があります。  
実際に簡単なHTMLを書いたりしたこともありましたが、  
CSSやJavaScriptについては「ようわからん……。」というレベルです。

## HTMLとは

HTML（Hyper Text Markup Language） は、  
Webサイトを構築する際に使われるマークアップ言語です。  

### 役割

HTMLの主な役割は**Webサイトの構造化**です。  
「構造化」といっても何をしているんだ、という感じですが、  
例えばこのページであれば、**ウィンドウタイトル**や**ヘッダー・フッターの定義**、  
**本文の段落構成**など、**何をどこに配置するのか**を決定することだと考えればよさそうです。  

ロードマップでは少し先になりますが、HTMLと組み合わせてWebサイトを構成する言語として以下の2つがあります。  
- CSS
- Javascript

CSSは**見た目の定義**、JavaScriptは**動きの定義**という役割分担になっています。  

## HTMLの基本構造

それでは簡単にHTMLの概要を整理したところで、基本構造について見ていきましょう。  
以下はSairy HubトップページのHTMLファイルの抜粋です。  

```html
<!DOCTYPE html>
<html lang = "ja">

<!-- ページ全体の定義 -->
<head>
  <meta charset = "UTF-8" />
  <title>Sairy Hub</title>
</head>

<body>
  <main>
    <!-- 本文 -->
  </main>
</body>

</html>
```

上から順番に見ていきましょう。  

### <!DOCTYPE html>

HTMLファイルの先頭に記述する**ドキュメントタイプ宣言**です。  
いろいろな言語で先頭におまじない的に記述する文言があると思いますが、あれですね。  
この記述は「このファイルはHTML5で記述されていますよ～」ということを宣言しています。  
なくても動くようですが、ブラウザの解釈が変わり、レイアウト崩れなどが発生したりするようなので、  
基本的には記述しておくのがよさそうです、本当におまじないですね。  

### <html lang = "ja">

HTML文書の開始と終了を示すタグです。  
あわせて属性としてlang="ja"を指定することで、  
「この文書は日本語で書かれていますよ～」という情報も伝えています。  

Sairy Hubでそれほど重視するつもりではないのですが、  
SEOにも影響するようで、意識しておくとよいポイントとなります。  
これもおまじない的に書いておけばいいんじゃないかと思います。  

### <head>

**メタデータの情報**を記載する範囲を指定するタグです。  
コンテンツとして表示するものというより、内部的なデータとして定義するものが中心となります。

### <body>

**コンテンツ**を記述する範囲を指定するタグです。  
\<head>タグは裏方、\<body>たぐは表舞台、のイメージでいいと思います。  

### <main>

コンテンツのうち、**主要となる範囲**を指定するタグです。  
この記事を書くにあたって少し調べてみましたが、  
他のページにも存在する繰り返し、使いまわし的な要素をここに含むのは不適切みたいです。  
そういう風に意識していませんでしたが、  
「変更したときの修正が大変そうだから」という理由でヘッダー・フッターは\<main>とは別のところに書いていたので、  
無意識のうちに適切な範囲を指定できていたみたいです。  

### その他

**セマンティックタグ**というものが存在し、より詳細に構造化する際に活用できるようです。  
抜粋するときに省きましたが、**\<section>タグ**や**\<article>タグ**など、特に意識せずに使っているものもありました。  
今後もサイト自体はアップデートしていくつもりなので、改善していきたいですね。
