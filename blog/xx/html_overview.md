## 前提知識など

## HTMLとは

Hyper Text Markup Languageの頭文字をとってHTML。
Webサイトを構築する際によく使われる言語です。  

### 役割

主な役割は**Webサイトの構造化**です。  
構造化、といっても何をしているんだ、という感じですが、  
例えばこのページのウィンドウタイトルやヘッダー・フッターの定義、段落など、  
**何をどこに配置するのか**を決定することだと考えればいいのかな、と思っています。  

ロードマップでは少し先になりますが、HTMLと組み合わせてWebサイトを構成する言語が以下の2つです。  
- CSS
- Javascript

CSSは**見た目の定義**、JavaScriptは**動きの定義**がそれぞれの役割となります。  

## HTMLの基本構造

それでは簡単にHTMLとは何か、を整理したところで、基本構造についてまとめていきましょう。  
以下がSairy HubトップページのHTMLファイルの抜粋です。  

```html
<!DOCTYPE html>
<html lang = "ja">

<!-- ページ全体の定義 -->
<head>
  <meta charset = "UTF-8" />
  <title>Sairy Hub</title>
</head>

<body>
  <main>
    <!-- 本文 -->
  </main>
</body>

</html>
```

上から順番に見ていきましょう。  

### <!DOCTYPE html>

HTMLファイルの先頭に記述する**ドキュメントタイプ**の宣言です。  
いろいろな言語において、先頭におまじない的に記述する文言があると思いますが、あれですね。  
この記載自体は「このファイルはHTML5で記述されていますよ～」ということを宣言しています。  
なくても動くようですが、レイアウト崩れなどが発生したりするようですね。  

### <html lang = "ja">

\<html>タグに属性としてlang="ja"を指定したものです。  
意味はそのまま、「HTML文書はここからここまでですよ～」と「日本語で書かれていますよ～」を指定するタグです。  

### <head>

メタデータの情報を記載する範囲を指定するタグです。  
コンテンツとして表示するものというより、内部的なデータとして定義するものを記述するタグですね。

### <body>

コンテンツを記述する範囲を指定するタグです。  
\<head>タグに対して、Webサイトのコンテンツとして表示する内容を記述することになります。

### <main>

コンテンツのうち、主要となる範囲を指定するタグです。  
この記事を書くにあたって少し調べてみましたが、  
他のページにも存在する繰り返し、使いまわし的な要素をここに含むのは不適切みたいです。  
（記述が大変になるから、という理由で無意識のうちに適切な範囲を指定できていました。）

### その他

**セマンティックタグ**というものが存在し、より詳細に構造化する際に活用できるようです。  
抜粋するときに省きましたが、**\<section>タグ**や**\<article>タグ**など、特に意識せずに使っているものもありました。  
今後もサイト自体はアップデートしていくつもりなので、改善していきたいですね。
