今後は必要に応じてサイレント修正をしていくと思いますが、最低限の設定として。

### box-sizingの設定

これは幅と高さの計算方法を指定する設定ですね。  
デフォルトは**content-box（border、paddingを含めない）**となっていますが、  
表示媒体にあった画面出力にしたいので割合指定をできるようにしましょう。  
つまり、**border-box**を指定します。  
```css
* {
  box-sizing: border-box;
}
```

### フォントの設定

次はフォントの設定です。  
ここは好みでいいのかな、メイリオとSegoe UIを設定しておきます。
あと調べてみたらsans-serifも指定することが多いんですかね。
```css
body {
  font-family: meiryo, "Segoe UI", sans-serif;
}
```

### 最大サイズと中央寄せ

サイズは適当に調整しようかな、と思うので、とりあえず学習に使ったサイトで例示されていた1200pxにしておきましょう。  
そして中央寄せ、これも忘れずに。  
```css
.container {
  max-width: 1200px;
  margin: auto;
}
```

### 余白の指定

これは設定だけ入れておきましょう。  
この辺りもそのうち触ることになると思いますし。
```css
body {
  margin: 0;
  padding: 0;
}
```
