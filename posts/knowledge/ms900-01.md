国内外で広く活用されているMicrosoft 365について、基礎を押さえるのであればかつてはMS-900というMicrosoft 365の基礎資格がありました。
「かつて」と書いた通り、すでに廃止されている資格です。
ただ、Microsoft 365とは何か、ということだけではなく、**クラウドとは何か**を整理する入口としては今でもかなり良い内容だったので、自分用の再学習メモとして改めてまとめ直していきます。

ちなみに、純粋な後継ではないですが、AB-900というものが用意されているようです。
[試験 AB-900 の学習ガイド: Microsoft 365 Copilot およびエージェント管理の基礎](https://learn.microsoft.com/ja-jp/credentials/certifications/resources/study-guides/ab-900)
そのうちAB-900も受験しようと思っているので、そちらもそのうちまとめます。

今回はまず全体像だけ。

## クラウドとは

かなり雑に言うと、

> 必要なITリソースを、必要なときに、必要なだけ使う仕組み

という理解で十分だと思っています。

昔はサーバーや機器を自前で持つ「オンプレミス」が基本でしたが、クラウドではインターネット経由で利用します。

ざっくり比較するとこんな感じです。

|        | クラウド | オンプレミス |
| ------ | ---: | -----: |
| 初期費用   |  小さい |    大きい |
| 拡張性    |   高い |     低い |
| カスタマイズ | 制限あり |     高い |

万能ではないですが、**最初から全部持たなくていい**のが大きな特徴です。

## Microsoft 365とは

Microsoft 365は単なるOffice製品ではなく、仕事を進めるためのサービス群をまとめたものです。

よく名前を見るものだと、

- Teams → チャット・会議
- SharePoint → ファイル共有
- OneDrive → 個人ストレージ
- Outlook → メール・予定表
- Word / Excel / PowerPoint → Office製品

あたり。

基本を押さえる、というところで言うとまずは

> Teamsを中心にいろんなサービスが繋がっているもの

くらいで理解できていればいいのではないかな、と思います。

## プランについて

大きく分けると、

- Business → 中小規模向け
- Enterprise → 大規模向け

という感じ。

さらに上位プランになるほど、

- セキュリティ
- 端末管理
- ID管理

が強くなっていきます。

目立つところで言うとOffice製品の有無が目につきますが、実際は「管理機能の差」の方がかなり大きい印象です。

## まとめ

- クラウドは「サーバーがどこかにある」ではなく利用形態の話
- Microsoft 365 ≠ Office製品
- プラン差はアプリより管理機能の方が重要

この辺りはご存じの方も多いですかね、クラウドとは何かは他の基礎試験でもよく問われます。
クラウドサービスが盛んな時代になっているので、是非とも押さえておきたいですね。
